Detailed Description

This class schedules tasks.

Note
This class is concret through but actually design as an abstract class. The default implementation of this class is naive that runs scheduled tasks immediately.

The implement of this class should run the scheduled task in any way, e.g. a background thread or defer until event loop is free, and finally notify that if the task finished or failed
